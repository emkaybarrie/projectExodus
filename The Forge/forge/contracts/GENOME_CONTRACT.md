# Genome Contract

Status: Canonical (FCL v1)
Last Updated: 2026-01-29
Scope: Defines artifact classification, graduation ritual, and evolution governance

---

## Purpose

The Genome is the Forge's **artifact classification system**. It distinguishes between:

- **Stable** artifacts: Constitutional, proven, change-controlled
- **Experimental** artifacts: New, under observation, lighter review

The Graduation Ritual governs how artifacts move from Experimental to Stable.

---

## Design Principles

1. **Visibility** â€” Classification is visible, not hidden metadata
2. **Policy** â€” Clear rules, not complex scoring systems
3. **Evidence-Based** â€” Graduation requires observable proof
4. **Director-Approved** â€” No automatic promotions
5. **Non-Bureaucratic** â€” Lightweight process, real value

---

## Two-Speed Genome

### Stable Artifacts

Constitutional artifacts that have graduated to production stability.

**Characteristics:**
- Proven over observation period (14+ days)
- No critical regressions during observation
- Smoke tests passing
- No open alert-level Reflex warnings tied to artifact
- Director-approved for stability

**Change Requirements:**
- Changes require Work Order with regression review
- Verifier-Tester sign-off recommended
- Higher scrutiny on modifications

**Examples:**
- FORGE_KERNEL.md
- FORGE_STATE.md
- FORGE_CAPSULE.md
- All contracts in `/contracts/`
- Core playbooks in `/ops/`

### Experimental Artifacts

New features or changes under observation.

**Characteristics:**
- Less than 14 days since introduction, OR
- Pending graduation review, OR
- Explicitly marked experimental

**Change Requirements:**
- Changes still require Work Order
- Lighter review burden
- Encouraged iteration

**Examples:**
- New portal features (M3/M4)
- New workflows
- FCL v1 cognition layer (until graduated)

## Eligibility Calculation

An artifact is eligible for graduation when:

```javascript
eligible = (
  daysSinceIntroduction >= 14 &&
  observations.smokePass === true &&
  !hasAlertWarnings(artifactPath) &&
  !hasCriticalRegressions(artifactPath)
)
```

### Eligibility Reasons

| Reason | Meaning |
|--------|---------|
| `eligible` | All criteria met, ready for graduation |
| `observation_pending` | Less than 14 days since introduction |
| `smoke_failing` | Smoke tests not passing |
| `alert_warnings` | Open alert-level warnings tied to artifact |
| `regression_detected` | Critical regression recorded for artifact |

1. **No automatic promotions** â€” Director approval required
2. **No auto-enforced blocks** â€” Classification is advisory

The Genome enables safe self-improvement:

```
New feature introduced â†’ Experimental
        â†“
Observation period (14 days)
        â†“

        â†“

        â†“

        â†“
```

- Silent drift from experimental to assumed-stable
- Evolution without evidence
